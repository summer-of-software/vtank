Objects

The object layer in gardener is designed to overlay the terrain layer. Objects are static things such as barrels, shrubs, and rocks that decorate the landscape of a VTank map. Objects are not required in a map. When editing a map it is suggested that the user focus on objects after they have completed the terrain and before they start adding events.
