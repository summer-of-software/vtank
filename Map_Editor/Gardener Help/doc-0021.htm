Opening A Map

When it comes to opening a map in Gardener, there's two places to turn to. The user can go the the menubar and select file->open or the user can select the open button on the toolbar.


After this selection has been made the standard open file dialog will be displayed. You'll only be able to select .vtmap files. Once one is selected, click ok and then the map will be loaded on to the canvas for you editing needs.
