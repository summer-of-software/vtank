Events

The event layer in gardener is designed to overlay the terrain and object layers in the map. Events don't show up on the map in game. They are used to determine how a map is suppose to be used in certain situations. Events are things such as spawn points, king of the hill areas, and flag capture and spawn points. When editing a map it is suggested that the user focus on the terrain and objects first, before adding events to the map.

As of current there are six different event tiles to use in gardener. They are as follows:

The player spawn point is where a player's tank will start in the map when the game starts or where they will start from after dying.
King of the hill area is used in the King of the hill game mode. This area represents the hill where the player's tank must be in order to score points in king of the hill.
The Red flag spawn point is where the red flag will start in the map when the game starts or where the red flag will return to after it has been reset. It will also double as the blue flag capture point where the blue flag must be brought in order to score.
The Blue flag spawn point is where the blue flag will start in the map when the game starts or where the blue flag will return to after it has been reset. It will also double as the red flag capture point where the red flag must be brought in order to score.
The red team spawn area is where the red team will start in the map when the game starts or where individuals of the red team will start from after dying.
The blue team spawn area is where the blue team will start in the map when the game starts or where individuals of the blue team will start from after dying.
